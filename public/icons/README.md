# Luna App Icons

Due to binary generation constraints, the actual PNG icons are not included in this code bundle.

To complete the PWA setup, generate two icons and place them in this folder:
1. \`icon-192.png\` (192x192 pixels)
2. \`icon-512.png\` (512x512 pixels)

**Design spec:** 
Use a solid rose pink (\`#E85D9A\`) circle background. In the center, place a white silhouette of a crescent moon. Ensure the moon is clearly visible with good contrast.
